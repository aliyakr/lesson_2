
Памятка команд для проекта "Симуляция движения частиц"

1. Установка необходимых библиотек
-----------------------------------
pip install numpy matplotlib

2. Структура проекта
--------------------
/project/
    /utils/
        classes.py       # Классы Particle и ParticleSimulator
        functions.py     # Функции для визуализации и генерации позиций
    main_project.py       # Главный файл запуска проекта
    instructions.txt      # Этот файл с памяткой

3. Как запустить проект
------------------------
Перейдите в папку проекта в терминале/командной строке:

cd /путь/до/project

Запустите главный файл:

python main_project.py

4. Дополнительно
----------------
- В файле main_project.py можно настроить:
    - Количество частиц (num_particles)
    - Количество шагов симуляции (num_steps)
    - Размер поля (bounds)
    - Цвет фона для визуализации (background_color)
    - Включение/отключение столкновений (enable_collisions)

- Для сохранения графика вместо показа раскомментируйте строку:
    save_plot(particles, background_color=background_color, filename='trajectories_output.png')

5. Примечание
--------------
Проект ориентирован на демонстрацию базовых физических моделей движения с столкновениями.
